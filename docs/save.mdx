### Save

Save `.md` files (and `formula.yml`) for a formula into the local registry. Works with a named formula, a source directory, or both.

```bash
g0 save <formula-name> [directory] [version-type]
```
- **version-type**: `stable` (optional)

### Quick examples

- **Save from directory into named formula**
```bash
g0 save my-notes ./docs
```

- **Save existing formula (discover files from its folder)**
```bash
g0 save my-notes
```

- **Save a directory only (auto-name from folder basename)**
```bash
g0 save /absolute/path/to/docs
```

- **Set an explicit version**
```bash
g0 save my-notes@1.2.3
```

- **Force overwrite an existing version**
```bash
g0 save my-notes ./docs --force
```

- **Bump versions**
```bash
# Prerelease bump (default when using -b)
g0 save my-notes -b patch

# Promote/bump to stable
g0 save my-notes stable -b minor

# Convert local prerelease to stable without bump
g0 save my-notes stable
```

- **Include dependencies**
```bash
g0 save app-docs --include ui-kit core-lib
g0 save app-docs --include-dev lint ci
```

### Options
- **-f, --force**: overwrite an existing version / skip confirmations
- **-b, --bump <type>**: `patch` | `minor` | `major`
  - Creates a prerelease by default; use with `stable` to create a stable bump
- **--include <names...>**: add formulas to this formula's `formulas`
- **--include-dev <names...>**: add formulas to `dev-formulas`

### Behavior
- Auto-creates `formula.yml` for a new formula and prints name/version
- Discovers `.md` files, updates frontmatter with the formula name, and resolves duplicates by latest modified time
- Saves files to the local registry and prints the saved file list
- Smart versioning:
  - New formulas start at a local prerelease of `0.1.0`
  - Subsequent saves auto-increment local prereleases
  - With `stable`, converts local versions to stable or bumps the stable version

### Sample output
```bash
✓ Found existing formula my-notes@0.1.0-local.3
✓ Bumping to prerelease version: 0.1.0-local.3 → 0.1.1-local.1
✓ Saved my-notes@0.1.1-local.1 (3 files):
   ├── formula.yml
   ├── README.md
   ├── docs/intro.md
```
